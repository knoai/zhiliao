# 知了云文档 - 本地启动说明

## 环境要求

- Python 3.10+
- Node.js 18+
- 本地 PostgreSQL 已配置（localhost:5432）

## 快速启动（推荐）

```bash
cd /Users/yu/Documents/cloud-notes/zhiliao-cloud-docs

# 一键启动前后端
./start-local.sh
```

## 手动分步启动

### 1. 后端启动

```bash
cd backend

# 创建虚拟环境（首次）
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate

# 安装依赖（首次）
pip install -r requirements.txt

# 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 前端启动

```bash
cd frontend

# 安装依赖（首次）
npm install

# 启动开发服务器
npm run dev
```

## 常见问题

### 1. 虚拟环境激活失败

```bash
# 检查 Python 版本
python3 --version

# 重新创建虚拟环境
rm -rf backend/venv
cd backend
python3 -m venv venv
source venv/bin/activate
```

### 2. 数据库连接失败

检查本地 PostgreSQL 容器是否运行：
```bash
docker ps | grep postgres
```

如果未运行，启动本地数据库：
```bash
docker run -d \
  --name cloud-notes-postgres \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_PASSWORD=postgres123 \
  -e POSTGRES_DB=cloud_notes \
  -p 5432:5432 \
  -v /Users/yu/Documents/u02/postgresql:/var/lib/postgresql/data \
  --restart unless-stopped \
  swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/postgres:15-alpine3.22
```

### 3. 端口被占用

修改启动端口：
```bash
# 后端换端口
uvicorn app.main:app --reload --port 8080

# 前端换端口
npm run dev -- --port 3000
```
